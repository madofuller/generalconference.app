## Troubleshooting

### "Failed to execute code" error

**Problem**: The Python API server isn't running

**Solution**:
1. Make sure Terminal 1 is running `python code_executor.py`
2. Check that you see "Server ready at http://localhost:5001"
3. Test with: `curl http://localhost:5001/health`

### "Gemini API key not configured" error

**Problem**: Environment variable not set

**Solution**:
1. Make sure `.env.local` file exists in `conference-app/`
2. File should contain: `GEMINI_API_KEY=AIzaSy...`
3. Restart the Next.js server (Ctrl+C, then `npm run dev`)

### API server can't find data

**Problem**: CSV file not in the right location

**Solution**:
1. Make sure `conference_talks_cleaned.csv` is in `conference-app/public/`
2. Check the path in `api_server/code_executor.py` if needed

### Port 5000 already in use

**Problem**: Another app is using port 5000

**Solution**:
1. Stop the other app
2. OR edit `code_executor.py` to use a different port
3. Update `conference-app/app/api/ask/route.ts` with the new port

## Technical Details

### Architecture

```
User Question
    ↓
Next.js API Route (/api/ask)
    ↓
Gemini API (generates Python code)
    ↓
Code Executor (localhost:5001)
    ↓
Pandas + DataFrame analysis
    ↓
Results + Charts returned
    ↓
Beautiful UI display
```

### Available Data

The AI has access to all conference talks with these columns:
- Basic info: title, speaker, calling, year, season, url
- Content: talk text, footnotes
- AI analysis: topics, emotions, scores
- Parsed JSON: topics/emotions arrays

### Code Generation

Gemini generates Python code like:
```python
# Count talks by speaker
result = df['speaker'].value_counts().head(10)
print(result)
```

The code is executed safely and results are returned to the UI.

## Security

⚠️ **Important Security Notes**:

- The API server runs **locally only** (localhost:5001)
- Code execution is **sandboxed** (limited functions)
- **Never expose** the API server to the internet
- Only run code generated by **your own Gemini API key**

The sandbox prevents:
- File system access
- Network requests
- Dangerous imports
- System commands

## Example Conversation

**You**: "Which speaker has given the most talks?"

**AI**: Generates code like:
```python
result = df['speaker'].value_counts().head(10)
```

**You see**:

**You**: "Show me talks by Thomas S. Monson about faith"

```python

result = faith_talks[['title', 'year', 'season']]
```

**You see**:
- Filtered list of talks
- Title, year, and season for each
