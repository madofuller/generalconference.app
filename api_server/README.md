# 🤖 AI Code Execution Server

This server executes Python code generated by AI to analyze conference talks data.

## Quick Start

### 1. Install Dependencies

```bash
cd api_server
pip install -r requirements.txt
```

### 2. Start the Server

```bash
python code_executor.py
```

The server will start on `http://localhost:5001`

### 3. Verify It's Running

```bash
curl http://localhost:5001/health
```

## How It Works

1. **Next.js app** sends user questions to `/api/ask`
2. **Gemini API** generates Python code to answer the question
3. **This server** executes the code safely
4. **Results** are returned with tables and charts

## API Endpoints

### `GET /health`
Health check endpoint.

**Response:**
```json
{
  "status": "ok",
  "rows": 4122
}
```

### `GET /data-info`
Returns dataset schema and sample data.

**Response:**
```json
{
  "shape": [4122, 18],
  "columns": ["title", "speaker", "calling", ...],
  "dtypes": {...},
  "sample": [...]
}
```

### `POST /execute`
Executes Python code and returns results.

**Request:**
```json
{
  "code": "result = df['speaker'].value_counts().head(10)"
}
```

**Response:**
```json
{
  "success": true,
  "output": "...",
  "data": {
    "type": "series",
    "data": {...}
  },
  "charts": []
}
```

## Safety Features

- **Sandboxed execution**: Limited to safe builtins
- **No file I/O**: Can't read/write files
- **No network access**: Can't make HTTP requests
- **Limited imports**: Only pandas, matplotlib, json
- **Copy of data**: Works on a copy of the DataFrame

## Available in Code

When your code runs, these variables/libraries are available:

- `df`: pandas DataFrame with all conference talks
- `pd`: pandas library
- `json`: json library
- `plt`: matplotlib.pyplot

## Example Code

### Count talks by speaker
```python
result = df['speaker'].value_counts().head(10)
print(result)
```

### Filter and group
```python
faith_talks = df[df['talk'].str.contains('faith', case=False, na=False)]
result = faith_talks.groupby('year').size().reset_index(name='count')
print(f"Found {len(faith_talks)} talks")
```

### Parse JSON columns
```python
import json

all_emotions = []
for emotions_str in df['emotions'].dropna():
    try:
        emotions = json.loads(emotions_str)
        all_emotions.extend(emotions)
    except:
        pass

from collections import Counter
result = dict(Counter(all_emotions).most_common(10))
```

### Create visualizations
```python
import matplotlib.pyplot as plt

# Group by year
yearly = df.groupby('year').size()

# Create plot
plt.figure(figsize=(12, 6))
plt.plot(yearly.index, yearly.values)
plt.title('Conference Talks Over Time')
plt.xlabel('Year')
plt.ylabel('Number of Talks')
plt.grid(True)

# Don't call plt.show() - the server handles it
# Just create the plot and it will be captured
```

## Troubleshooting

### Server won't start
- Make sure you've installed dependencies: `pip install -r requirements.txt`
- Check that the CSV file exists at the correct path
- Make sure port 5000 is not in use

### Code execution fails
- Check the error message in the response
- Verify your code syntax
- Make sure you're using only allowed libraries
- Store your final result in the `result` variable

### Charts not appearing
- Make sure you create figures with `plt.figure()`
- Don't call `plt.show()` - the server captures plots automatically
- The server will return charts as base64-encoded PNG images

## Security Notes

⚠️ **This server executes arbitrary Python code!**

- Only run this server locally
- Never expose it to the internet
- The sandbox provides basic protection but is not foolproof
- Only execute code you trust or that's generated by your own AI

## Dataset Schema

The `df` DataFrame has these columns:

- `title`: Talk title
- `speaker`: Speaker name
- `calling`: Speaker calling/position
- `year`: Year (integer, 1971-2025)
- `season`: "April" or "October"
- `url`: Link to talk
- `talk`: Full text
- `footnotes`: Footnotes text
- `topics`: JSON array of topic labels (from NLP)
- `topic_scores`: JSON array of scores
- `primary_topic`: Top topic
- `primary_topic_score`: Score for top topic
- `emotions`: JSON array of emotion labels (from NLP)
- `emotion_scores`: JSON array of scores
- `primary_emotion`: Top emotion
- `primary_emotion_score`: Score for top emotion
- `all_emotion_scores`: JSON object with all scores
